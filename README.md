# DPLE
Design Patterns And Living Examples: 什么是设计模式，开源软件如何用设计模式，项目中如何用设计模式及实例


# Milestone
- [ ] 深入分析每种设计模式
- [ ] 著名开源中的: 如React全家桶相关 | VSCode | Bable | ESBuild | NodeJS | ES6 | FastAPI | Nginx | Git
- [ ] 日常项目(以往代码)中，哪些场景可以使用设计模式，甚至以此为依据进行重构

- [ ] 开源中的设计模式
